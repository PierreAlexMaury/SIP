Questions

• What type of protocol is SIP?

SIP is a text based protocol as HTTP. It gives flexibility and facility to communicate and work with HTTP and SMTP

• What are the primitives defined into RFC 3261?

INVITE, ACK, BYE, CANCEL, OPTIONS and REGISTER

• What does a location server do?

It obtains informations about the location of the user after it has received registration requests from the UA.

• What is a SIP URI? What is its main advantage?

It is a generic identifier used to name a SIP user on the Internet. It has a format based on e-mail address format and it is unique for a user so we can make free calls to all other users in the world by simply composing their SIP addresses.

• In what application Apple uses SIP? another example?

Apple uses SIP for its application FaceTime and for example Google uses SIP for Google voice
